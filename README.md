# PxDimenFactory

**What can it does？**

选择分辨率并自动生成对应的dimen.xml文件。

Select the resolution and automatically generate the corresponding dimen.xml files.

**How to use？**

clone项目，然后运行PxDimenBuilder下的Main.java即可。

just clone and run the Main.java in PxDimenBuilder.
